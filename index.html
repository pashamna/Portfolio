This project focuses on Speech Emotion Recognition (SER) using deep learning techniques, specifically Convolutional Neural Networks (CNNs) to analyze emotional cues from speech. I utilized the Toronto Emotional Speech Set (TESS) dataset and employed Python, TensorFlow, and Keras for model development, with signal processing techniques for feature extraction and enhanced classification accuracy.

Virtual Study Space

The Virtual Study Space is a web app enhancing productivity with customizable study environments, a Pomodoro timer, task tracking, and group study via WebRTC video calls. Built with HTML, CSS, JavaScript, PHP, and Firebase, it lets students personalize backgrounds, manage tasks, and collaborate seamlessly. This platform promotes focus and combats procrastination, creating an engaging virtual learning space.
